# 🚀 Jan Gustavo · Portfolio Web

Este é o repositório do meu portfólio pessoal, acessível em [jangustavo.me](https://jangustavo.me/). O objetivo deste projeto é demonstrar minhas habilidades em **Desenvolvimento Backend**, **Frontend Moderno** e **Boas Práticas de Engenharia de Software**.

O design foi concebido com uma estética de terminal, reforçando meu foco em **Backend**, **Sistemas Operacionais** e **Automação**.

---

## 🛠️ Stack Tecnológica

*   **Frontend:** React.js / Vite (ou sua tecnologia atual, ex: HTML/CSS/JS puro)
*   **Estilização:** Tailwind CSS / CSS Modules (com foco em performance e responsividade)
*   **Hospedagem:** GitHub Pages
*   **Domínio:** Cloudflare (DNS e Segurança)
*   **CI/CD:** GitHub Actions (Automação de Deploy)

---

## ✨ Funcionalidades e Diferenciais

*   **Design Terminal-Style:** Interface minimalista e temática, otimizada para desenvolvedores.
*   **Responsividade Total:** Experiência fluida em dispositivos móveis, tablets e desktops.
*   **SEO Otimizado:** Meta tags estruturadas para melhor ranqueamento em motores de busca.
*   **Performance:** Carregamento rápido com ativos otimizados e código limpo.
*   **Integração Contínua:** Deploy automatizado via GitHub Actions a cada novo commit na branch `main`.

---

## 📂 Estrutura do Projeto

```bash
├── public/          # Ativos estáticos (ícones, imagens)
├── src/             # Código-fonte da aplicação
│   ├── components/  # Componentes reutilizáveis
│   ├── styles/      # Arquivos de estilização (CSS/Tailwind)
│   └── data/        # Arquivos JSON/JS com dados dos projetos
├── .github/         # Workflows de CI/CD (GitHub Actions)
└── index.html       # Ponto de entrada da aplicação
```

## 🚀 Como Rodar Localmente

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/JanGustavo/JanGustavo.github.io.git
    ```
2.  **Instale as dependências (se aplicável):**
    ```bash
    npm install
    ```
3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
